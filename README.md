# 🎬 YouTube Video Analyzer

An AI-powered YouTube video analysis tool built with **Agno** and **Groq LLM** that extracts transcripts and generates detailed insights from any YouTube video — instantly.

---

## 🚀 Features

- 🔍 **Transcript Extraction** — Automatically fetches transcripts from any YouTube video
- 🤖 **AI-Powered Analysis** — Uses Groq's LLaMA models to analyze and summarize content
- 📊 **Detailed Reports** — Generates structured analysis reports from video content
- 🖥️ **Clean UI** — Simple and intuitive interface built with Streamlit
- ⚡ **Fast & Lightweight** — Powered by Groq's ultra-fast inference engine

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| AI Framework | [Agno](https://github.com/agno-agi/agno) |
| LLM Provider | [Groq](https://groq.com) (LLaMA 3) |
| Transcript | youtube-transcript-api |
| UI | Streamlit |
| Language | Python 3.11 |
